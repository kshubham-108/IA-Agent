# Corpus

Vision cards and reference data for extraction evals and calibration (PROJECT_BRIEF §5b).

## Contents

| Path | Description |
|---|---|
| `vision-cards/` | Vision card documents (docx/pdf) |
| `vision-cards-parsed/` | Plain-text extracts (generated by ingest) |
| `vision-cards-index.json` | Metadata index for all vision cards |
| `wall_of_reference.json` | Calibrated Wall of Reference rows + actuals (generated from xlsx) |
| `Wall of Reference - Calibrated Projects.xlsx` | Source spreadsheet |
| `Baselining - Project Wall of Reference v0 1.pdf` | Wall of Reference reference PDF |
| `labelled-pairs.json` | Four Vision Card ↔ BA score eval pairs (§5b) |
| `scripts/ingest.ts` | TypeScript corpus ingestion (`pnpm ingest-corpus`) |
| `scripts/ingest.py` | Python fallback ingestion (no Node required) |

## Ingestion

```bash
pnpm ingest-corpus
# or: python3 corpus/scripts/ingest.py
```

Produces `wall_of_reference.json`, `vision-cards-index.json`, and `vision-cards-parsed/*.txt`.

## Labelled eval pairs

See `labelled-pairs.json`. Source files live in `vision-cards/`.
