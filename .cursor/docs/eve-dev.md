# Eve framework documentation

Agents working on `agent/` should consult the Eve docs:

- **URL:** https://eve.dev/docs
- **Cursor @Docs:** Settings → Features → Docs → add `https://eve.dev/docs` (or `eve.dev`) so `@Docs eve.dev` works in chat.

There is no programmatic way to register @Docs from the repo; this file serves as a reminder until indexed manually.
